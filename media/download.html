Steps to download the site.

  1. Firefox should be prompting you to 'store data on your computer for offline use' above. Choose Allow.
  2. Any popups that prompt you to download, click Cancel (and alert the system administrator).
  3. Stay on this page until this box turns green:
    (or until it stops saying 'Transferring ' in the status bar at the bottom).
  4. Bookmark this page: Masivukeni Start
  5. Go to the Firefox File menu, and choose 'Work Offline.'
  6. Visit the bookmark, and try out the application offline.
