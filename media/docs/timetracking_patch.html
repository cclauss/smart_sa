Time-Tracking Patch installation

1) Get the patch

Get it from here. Unzip it onto a USB key. It should result in a directory named "patch_timetracking_2013".

Copy on C:

Plug the USB key into the laptop, open it for browsing.

Open another window and go to "C:".

Copy the "patch_timetracking_2013" directory to C:

Install

Open up the newly copied "C:\\patch_timetracking_2013\" directory and double-click on "install". A terminal window should open up and some text should scroll by. Give it a few minutes to run.

It will close the terminal window itself when it is finished.

Reboot and test

Reboot the computer. Log into the Masivukeni app as usual, and make sure it appears to be working as normal.

At this point, it is safe to delete the "C:\\patch_timetracking_2013" directory (though not necessary).
