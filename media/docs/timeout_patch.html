Timeout Patch installation

1) Get the patch

Get it from here. Unzip it onto a USB key. It should result in a directory named "patch_timeout_2014".

Copy on C:

Plug the USB key into the laptop, open it for browsing. Open up "C:" in another window and copy the "patch_timeout_2014" folder there.

Run installer

Go into the "C:\patch_timeout_2014\" folder, right click on the "install" file and select "Run as Administrator".

Restart Webserver

Running that installer changed a config file for the webserver. Now it needs to be restarted for the change to take effect. This step is actually optional; you could instead just reboot the machine and will be fine. Doing it this way is a little faster though.

Run the Bitnami Manager tool. You can find it under "Bitnami DjangoStack" in the start menu, or search for "Manager Tool". It will look like this:

Select "Apache Web Server" and hit the "Restart" button on the right.

That should do it. The webserver's timeout has now been changed from 5 minutes to 50 minutes, which ought to be long enough...
