Installation

Preparation

Download the latest install zip from http://slank.ccnmtl.columbia.edu/masivukeni_installer.zip and unzip it to a USB thumb drive. It should contain a "smart_sa" folder, a "bitnami-djangostack-..." installer and a "ChromeStandaloneSetup" which can be used to install the Chrome browser if the computer doesn't already come with it (which it should).

If you are replacing a broken or lost clinic laptop, you should also download the most recent backup file for that clinic from the online site.

Netbook Installation

Insert the USB drive with the install media. Open it for browsing.

Open up another explorer window

Copy the "smart_sa" folder from the USB drive to "C:"

Bitnami Install

Bitnami is a package that provides a lot of the base technologies that Masivukeni relies on.

Copy the "bitnami-djangostack..." installer program to the desktop (or anywhere else you can find it easily)

Double-click it to run the Bitnami installer

Uncheck "MySQL", but leave the others checked.

Leave the default installation directory

When it prompts for the PostgreSQL password, enter "masivukeni" in both fields

Let it associate .py files with the bundled Python

Do not set up a default project

Uncheck the "learn more about Bitnami Cloud Hosting" checkbox. That will just try to open a browser to their site.

Similarly, uncheck "Launch Bitnami DjangoStack"

Masivukeni Install

Go back to "C:" and open up "smart_sa"

Right click on "admin_initialize" and select "Run as Administrator". It will open up a terminal window and a bunch of stuff will start scrolling by.

When that has finished (which could take a while), double click "initialize". This will open another terminal window.

It will pretty quickly ask you for a password though. Enter "masivukeni" as the password here

And again here:

A lot of messages will scroll by. A few will look like errors. Particularly, here:

Don't panic! That's ok. It'll keep going.

When it has finished, select "Manager tool" from the Windows Start Menu. Select "Apache" and hit the "Restart" button:

The Masivukeni app should now be running and ready to connect to with a browser. Open up Chrome and type "http://localhost/" in the URL bar. You should see the intro page for the site:

Login as the admin user (you should already know the standard default admin password)

Go into "Administrate" and set the laptop's "Clinic Name" (and hit "Save"). That will allow us to connect the data from the laptop to the correct clinic later.

Restore Data from backups

If you are replacing a broken laptop, go down to "Restore Participant Data" and upload the most recent backup file (which you can get from the online site)

Set it up to start automatically

Click the wrench icon in the top-right of Chrome and select "Options". Under "Home Page", make sure "Open This Page:" is selected, and type in "http://localhost/". That will make sure that Chrome goes directly to Masivukeni each time it's launched.

Now, to make Chrome start up automatically when Windows starts, open the Windows Start Menu, Click "All Programs", scroll down so you can see the "Startup" folder. Right click it and select "Open".

Drag a Chrome shortcut icon in there from the Desktop or Application folder.

For a Clinic laptop, it's probably also a good idea to clean things up a bit and delete all the extraneous icons from the Desktop and the toolbar. Maybe rename the Chrome icon to "Masivukeni" to make it as obvious as possible.

Congratulations! You're done. Then next time this computer is started, Chrome should start up and go directly to Masivukeni.
