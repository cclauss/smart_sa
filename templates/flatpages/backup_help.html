Instructions on installing Masivukeni on a laptop from scratch

  1. copy Masivukeni.zip to your My Documents folder.
    • open the file (double click on the folder with the zipper)
    • click the 'extract all files' link on the left (Mac behavior may be slightly different)
    • answer yes to all the defaults. If it asks if you want to overwrite a file, click 'Yes to All'
      • this should place the extracted copy in C:/Documents and Settings/Administrator/My Documents/Masivukeni/Masivukeni/
  2. Open Firefox (in order to change settings)
    • In the address bar, type: 'file://localhost/C:/Documents and Settings/Administrator/My Documents/Masivukeni/Masivukeni/home.html'
    • From the menu, click Tools->Options.
    • Set to the Default Homepage:
      • Click the Main tab, and then click the button 'Use Current Page.'
    • Do not remember form data (for privacy reasons across users)
      • Click the Privacy tab, and then UNCHECK 'Remember what I enter in forms and the search bar'
    • Click OK
  3. Login as admin and click Create Backup.
    • The first time it will give a security warning, confirm, and check 'make permanent'--this is to make it possible to open the Save As window for the backup file.
  4. Follow the restore procedure below to restore accounts and data.

To update your Masivukeni software (while keeping data)

This scenario is if we need to update the tasks or content, after the site has been deployed.

  1. CCNMTL will provide a Masivukeni.zip file. This file will be the same as the one described above
    • EXCEPT that it will not include the video directory contents (for file size reasons)
  2. BACKUP your data a final time before updating
  3. Follow step #1 above (in installing Masivukeni from scratch)
  4. Look in the directory My Documents/Masivukeni/Masivukeni/multimedia/video/
    • If you see several video files there, then you are finished
    • If there are no files in that directory, copy the files in My Documents/multimedia/video/ to this folder

To Restore data to a newly installed version of Masivukeni

  1. (from a CD or thumbdrive) copy the backup file to your My Documents folder. Rename this file restore.html.

or

  1. (from an online download) Go to http://smartsa.ccnmtl.columbia.edu.
    • click back up and restore client data
    • Look for the list of available backups. click the name of the most recent backup to download to your desktop
    • This file will be named restore.html. Put it in your My Documents folder.
  2. Open Masivukeni as usual. Do not log in. Click the 'restore' link. This will open the restore file. It is necessary to open this file once before going through the rest of the restore procedure.
  3. Click the link to 'Masivukeni Administrator Log In'
  4. log in with the administrative password (call CCNMTL to get this password)
  5. Look for Restore From Backup and click the button that says 'show available backups'. You should see a link that includes today's date. Click this link. The restore process will now begin. This may take up to two minutes. A message will appear after the restoral is complete. Do not leave the page, or close the browser until it is finished.

To erase corrupted data and restore from a recent backup

  1. go to http://smartsa.ccnmtl.columbia.edu.
    • click back up and restore client data
    • Look for the list of available backups. click the name of the most recent backup to download to your desktop
    • This file will be named restore.html. Put it in your My Documents folder.
  2. open Masivukeni as usual
  3. do not log in. click 'go to restore.html fil in 'My Documents' to restore from a backup'

This will open the restore file. It is necessary to open this file once before going through the rest of the restore procedure.

  1. Click the link to 'Masivukeni Administrator Log In'
  2. log in with your password
  3. Look for Restore From Backup and click the button that says 'show available backups'. You should see a link that includes today's date. Click this link. The restore process will now begin. This may take up to two minutes. A message will appear after the restoral is complete. Do not leave the page, or close the browser until it is finished.

To create and upload a backup

  1. log into Masivukeni
  2. look for 'File Backup' and click 'Create Backup File.'
  3. The document save dialog box will open. Leave the default name for the file. Note where it is being saved and click save.
  4. when you are connected to the intenet, go to http://smartsa.ccnmtl.columbia.edu
  5. click back up and restore client data
  6. click 'browse' and navigate to your backup file
  7. click submit
  8. when the file transmission is complete a message will appear at the top of the screen that says 'Successfully saved the backup file'
