Save this file

From the File menu, click Save Page As and make sure the file location is on your portable drive which you will upload from the computer with an Internet connection. It will save as restore.html but you can change this to something else.

If for some reason, you need to restore a backup, put this file in the My Documents folder naming it restore.html, and run the Restore shortcut from your Desktop. Then you can go into the admin interface and restore from there.
