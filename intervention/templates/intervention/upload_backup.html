Instructions for restoral

  1. Save the appropriate download here as restore.html
  2. Copy it to the laptop, putting it in your "My Documents" folder. NOTE: You may have to overwrite the current file there
  3. Do NOT open the file directly
  4. Instead, on your desktop, there is a Restore Masivukeni Data icon on your Desktop. Double-click that icon.
  5. Once the restoral file opens, open Masivukeni from the normal desktop icon.
  6. Go to the admin page, and run the restoral
